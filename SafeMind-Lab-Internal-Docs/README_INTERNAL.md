# SafeMind Lab – Internal Documentation

This repository contains internal-only founding, product, and technical documents.
These documents are intended to align the founding team and evolve over time.

## Reading Guide
- 00_Founding_Archive: Long-lived truths and founding principles
- 01_Product: Product hypotheses and definitions
- 02_Technical_Architecture: System and protocol designs
- 03_Market_&_Strategy: Market interpretations (non-binding)
- 04_Risks_&_Open_Questions: Known risks and unknowns
- 05_Roadmaps: Planning hypotheses

## Document Status
Each document should clearly state whether it is:
Draft / Hypothesis / Experimental / Committed
